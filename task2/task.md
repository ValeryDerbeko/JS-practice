**Корректные скобки**

Дана строка `s`, содержащая только символы `'('`, `')'`, `'{'`, `'}'`, `'['` и `']'`. Определите, является ли входная строка корректной.

Строка считается корректной, если:

1. Открытые скобки должны быть закрыты скобками того же типа.
2. Открытые скобки должны закрываться в правильном порядке.
3. Каждая закрывающая скобка должна иметь соответствующую открытую скобку того же типа.

**Примеры:**

**Пример 1:**
Вход: `s = "()"`
Выход: `true`

**Пример 2:**
Вход: `s = "()[]{}"`
Выход: `true`

**Пример 3:**
Вход: `s = "(]"`
Выход: `false`

**Пример 4:**
Вход: `s = "([])"`
Выход: `true`

**Пример 5:**
Вход: `s = "([)]"`
Выход: `false`

**Ограничения:**

* `1 <= s.length <= 10^4`
* `s` состоит только из скобок `'()[]{}'`
