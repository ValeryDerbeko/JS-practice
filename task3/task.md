## Слияние двух отсортированных списков

Вам даны **головы двух отсортированных связанных списков** `list1` и `list2`.

Объедините эти два списка в **один отсортированный список**. Список должен быть составлен **путём «сцепления» узлов первых двух списков**.

**Верните голову** объединённого связанного списка.

![alt text](image.png)

---

### Примеры

**Пример 1:**

```
Вход: list1 = [1,2,4], list2 = [1,3,4]
Выход: [1,1,2,3,4,4]
```

**Пример 2:**

```
Вход: list1 = [], list2 = []
Выход: []
```

**Пример 3:**

```
Вход: list1 = [], list2 = [0]
Выход: [0]
```

---

### Ограничения

* Количество узлов в обоих списках находится в диапазоне `[0, 50]`.
* `-100 <= Node.val <= 100`
* Оба списка отсортированы в **неубывающем порядке**
