# JavaScript Practice

Репозиторий с решениями различных задач на JavaScript для практики и повышения качества кода. Задачи в основном берутся с платформы LeetCode.

В репозитории каждая задача хранится отдельно. Для задачи сохраняется её условие и одно или несколько решений.

Условия задач размещаются в файле `task.md`. Решения находятся в файле `solution.js`.

Для некоторых задач присутствует несколько вариантов решения. Например, сначала добавляется черновой вариант — рабочий, но не оптимизированный и трудночитаемый, затем решение улучшается.

Репозиторий носит учебный характер. Решения могут пересматриваться и улучшаться со временем.
